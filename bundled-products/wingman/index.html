Features

Built for the terminal. Built for Windows.

Everything you need to keep a Windows machine up to date — no web browser required.

Parallel Discovery

All package managers are scanned simultaneously. No sequential waiting — results appear in seconds.

FAST

CRT Terminal UI

An interactive TUI built with Bubble Tea. Keyboard-driven, filter, sort, and upgrade without leaving the terminal.

TUI

Non-Interactive Mode

Run wingman.exe -y in CI or scheduled tasks to upgrade everything that supports auto-update silently.

SMART

Safe by Default

Wingman shows you exactly what will change before touching anything. Preview mode first, commit when ready.

SAFE

Go-Powered Core

Written in Go 1.23+. A single statically linked binary — no runtime, no dependencies, no installer drama.

GO

Open Source & Extensible

MIT licensed. Add your own package manager source by implementing a simple Go interface. PRs welcome.

OPEN
How it works

From zero to upgraded in four steps

Windows 10/11 + Go 1.23+ is all you need. Everything else is optional.

01

Install Go and clone

Make sure Go 1.23 or later is on your PATH, then clone the Wingman repository from GitHub.

git clone https://github.com/piyushmishra318/wingman
02

Build or run directly

Use the provided batch file for a one-shot build, or run straight from source during development.

wingman.bat
03

Wingman scans everything

All detected package managers are queried in parallel. A live TUI table populates as results arrive — no spinner needed.

go run ./cmd/wingman
04

Select and upgrade

Navigate rows with arrow keys, press U for a single package or A for all updates. Or run fully unattended with -y.

wingman.exe -y

Your packages are waiting.
