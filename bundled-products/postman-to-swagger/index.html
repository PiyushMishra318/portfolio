How it works

From collection to docs in seconds

The converter parses your Postman collection structure and maps each request to an OpenAPI 2.0 path object.

Step 01

Paste or upload your collection

Export your Postman collection as JSON (v2.0 or v2.1) and paste it into the textarea, or use the file upload button.

collection.json
Step 02

Hit Convert

The UI sends a POST request to /convert. The NestJS serverless function parses the collection and builds the OpenAPI schema.

POST /convert
Step 03

Explore in Swagger UI

The response is immediately fed to Swagger UI. Every endpoint, method, parameter, and description is rendered interactively.

swagger.json
Step 04

Download or integrate

Copy the generated JSON or wire the /convert endpoint into your pipeline to keep docs in sync with your Postman workspace.

REST API

One endpoint. Dead simple.

POST a Postman collection JSON body to /convert and receive a Swagger 2.0 JSON object in response.

Request

POST /products/postman-to-swagger/convert/
Content-Type: application/json

// Body: your Postman collection JSON
{
  "info": { "name": "My API", ... },
  "item": [...]
}

Response — 200 OK

{
  "swagger": "2.0",
  "info": { "title": "My API", "version": "1.0" },
  "paths": {
    "/api/users": {
      "get": { "summary": "Get users", ... },
      "post": { "summary": "Create user", ... }
    }
  }
}
