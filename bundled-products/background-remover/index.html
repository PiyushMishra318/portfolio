01 / Overview

What it does

BackgroundRemover separates moving foreground objects from a static scene in real time. Each incoming frame is compared against an evolving background model — pixels that diverge significantly become the binary foreground mask.

Built as an OpenCV learning project, it demonstrates two state-of-the-art classical algorithms: MOG2 handles gradual lighting changes and shadow detection, while KNN provides robustness against sensor noise and brief occlusions. Both run at real-time speed on standard hardware without any GPU acceleration.

Computer Vision Background Subtraction Real-Time OpenCV C++ Shadow Detection

Shadow Detection

MOG2 distinguishes shadows from true foreground objects, reducing false positives in outdoor and mixed-lighting scenes.

Noise Robustness

KNN-based model handles sensor noise and brief occlusions better than simple frame differencing approaches.

Live VideoCapture

Reads from any camera or video file via cv::VideoCapture — swap the source without modifying the pipeline.

Real-Time Speed

Native C++ implementation with OpenCV's optimized routines processes frames at full rate with minimal latency.

02 / Pipeline

How it works

01

Capture

cv::VideoCapture opens camera index 0 or a video file path, reading frames in BGR format at the source's native framerate. The loop runs until the stream ends or the user presses q.

cv::VideoCapture cap(0);
cap >> frame;               // read next BGR frame
02

Model Update

The background subtractor's apply() method updates the internal model each frame and outputs a binary foreground mask. MOG2 models each pixel as a mixture of Gaussians; KNN tracks the K nearest background samples.

pMOG2->apply(frame, fgMask);   // or pKNN->apply(...)
// fgMask: 255 = foreground, 127 = shadow, 0 = background
03

Display

cv::imshow renders both the original frame and the foreground mask in separate named windows, updated each iteration at the camera's native framerate.

cv::imshow("Frame",   frame);
cv::imshow("FG Mask", fgMask);
cv::waitKey(30);
