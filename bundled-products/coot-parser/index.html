What It Does

FOUR
MODES.

Himalaya AST

Parse any HTML export into a full JSON abstract syntax tree using the Himalaya parser. Inspect every node, attribute, and text fragment programmatically.

React Markup

Convert raw HTML into JSX-friendly output: class becomes className, inline styles become objects, and DOM events map to React props.

Compose Pages

Feed a canvas JSON array of positioned components and get back a single composed HTML page with absolute positioning, pseudo-state CSS, and injected scripts.

Split Components

Mark layers with data-component attributes and the parser splits them into individual files, outputting a structured component directory ready for your project.

How It Works

PARSE
FLOW.

01

Export from Editor

Take any HTML export or canvas JSON array from your Canva-style design editor. No preprocessing required — paste or point the CLI at the file.

02

Choose a Mode

Run ast, react, compose, or split as your CLI subcommand. Each mode targets a specific output format.

03

Himalaya Parses

The Himalaya library tokenises your HTML into a typed AST. The parser then walks that tree, applying conversions and attribute transforms.

04

Output Written

Results are written to --out or --out-dir. Use the REST demo server for browser-based parsing with live preview tabs.

canvas-component-parser — bash 
# Install
$ git clone git@github.com:PiyushMishra318/canvas-component-parser.git
$ npm install

# AST — HTML to JSON tree
$ node bin/canvas-component-parser.js ast fixtures/template.html --out ast.json

# React — JSX-ready markup
$ node bin/canvas-component-parser.js react fixtures/template.html --out out.jsx.html

# Compose — canvas JSON to full page
$ node bin/canvas-component-parser.js compose fixtures/components.sample.json --out page.html

# Split — by data-component attribute
$ node bin/canvas-component-parser.js split fixtures/template.html --out-dir dist/

# Demo server
$ npm start
  Listening on http://127.0.0.1:3000

CLI Reference

ALL
COMMANDS.

ast

canvas-component-parser ast <file>

Produces a full Himalaya AST in JSON. Every element, attribute, and text node is captured. Pipe to jq or feed to downstream tooling.

react

canvas-component-parser react <file>

Rewrites HTML for JSX compatibility: class → className, style strings → JS objects, onclick → onClick, and more.

compose

canvas-component-parser compose <json>

Takes a JSON array of canvas layer objects with html, style, position, and scripts fields. Outputs a single composed HTML page.

split

canvas-component-parser split <file>

Walks the AST for elements bearing data-component="Name" and writes each as its own file into the output directory — one component per file.
