What it does

Everything you need, nothing you don't

Real-time Battery Monitoring

Precise tracking for both wired and wireless controllers via XInput. Percentage updates live so you always know where you stand before a long session.

Bluetooth Fallback

Custom PowerShell integration accurately reports battery levels when standard XInput is unavailable — perfect for Bluetooth-connected controllers.

Session Analytics

Track total play time and battery drain percentage per session. Understand exactly how your battery performs during different game genres and intensities.

Performance History

View a detailed log of past sessions to see how your battery sets are performing over time. Identify degradation trends before they disrupt gameplay.

Smooth Resizing

A fully responsive Electron UI that scales to any preferred desktop size. Drag a corner to resize the widget and it adapts instantly without layout jank.

Minimize to Tray

Runs as a lightweight background service. Double-click the system tray icon to show or hide the widget. Zero desktop clutter when you need your screen space.

Under the hood

How XBat works

  1. 01

    Connect Controller

    Plug in via USB or pair via Bluetooth. XBat detects your Xbox controller automatically on launch.

  2. 02

    XInput Query

    The Electron main process calls the native XInput API at 30 Hz to retrieve precise battery level and controller state.

  3. 03

    PowerShell Fallback

    When XInput returns no data (Bluetooth mode), a PowerShell script queries the WMI battery namespace for accurate readings.

  4. 04

    Live UI Update

    Data is piped via IPC to the renderer process and displayed in the glassmorphism widget with animated bar and session timer.
