WHAT MAKES IT TICK

Every technical choice was made for smooth, dependency-free gameplay across all devices.

HIGH-DPI CANVAS

Crisp rendering on Retina and HiDPI displays via devicePixelRatio scaling — no blurry sprites ever.

DESYNCHRONIZED CONTEXT

The desynchronized canvas hint bypasses the main thread compositor for minimal input latency.

SPRITE SHEET ANIMATION

Flappy Bird reads from a single packed PNG sprite sheet, cycling frames at 60 fps for buttery-smooth animation.

MOBILE JOYSTICK

Space Shooter ships an on-screen virtual joystick so touch-screen players get the full experience with no keyboard needed.

MEDALS & HIGH SCORES

Earn bronze, silver, and gold medals in Flappy Bird based on your score. High scores persist across sessions via localStorage.

ZERO DEPENDENCIES

Vanilla JavaScript throughout — no React, no bundler, no transpiler. Clone it and open index.html directly.

TECH STACK

BUILT WITH THE WEB PLATFORM

Every game runs on open web standards. No transpilation. No framework overhead.

  • HTML5 CANVAS API 2D rendering context with desynchronized hints and pixel ratio scaling
  • VANILLA JAVASCRIPT ES2020+ modules, requestAnimationFrame game loop, no transpiler required
  • TOUCH & KEYBOARD INPUT Unified input layer — keyboard events on desktop, virtual joystick on mobile
  • STATIC HOSTING Deployed on Vercel — no server, no database, instant global CDN delivery
