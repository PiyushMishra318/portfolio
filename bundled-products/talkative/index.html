Features

Everything you need to connect

Built with Flutter and Firebase, Talkative gives you the fundamentals of social messaging — clean, fast, and open source.

Google Sign-In

Authenticate instantly with your Google account. No passwords to remember — secure OAuth 2.0 flow with a single tap.

Clean Inbox

A focused conversation list that shows your active chats at a glance. No noise, no algorithm — just the people you actually talk to.

Real-Time Messaging

Send and receive messages instantly. Powered by Firebase Firestore with live updates — your conversation thread stays in sync.

Guest / Dev Mode

Skip OAuth setup during local development. Continue as guest to explore all UI screens without configuring credentials.

Secure Auth Flow

Industry-standard Google OAuth 2.0. Your credentials never touch our servers — Firebase handles authentication end-to-end.

Cross-Platform Ready

Built with Flutter — same codebase targets Android, iOS, and Web. One code, many platforms, zero compromise on experience.

How It Works

Up and running in minutes

From clone to conversation — get Talkative running locally or deploy your own instance with a few commands.

1

Clone the Repo

Fork or clone from GitHub. All source code is MIT licensed and fully open to explore.

2

Install Flutter

Install Flutter stable and run flutter pub get to fetch all dependencies in seconds.

3

Configure OAuth

Set up Google OAuth client IDs in Cloud Console, or use Guest mode to skip setup during dev.

4

Run & Chat

Launch with flutter run and start chatting. Run tests with flutter test && flutter analyze.
