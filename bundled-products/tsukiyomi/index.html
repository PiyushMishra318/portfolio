What it does

Everything your GBA library deserves

ROM Metadata Browser

Instantly browse 2 800+ GBA titles with full metadata — genre, region, release year, and description — served from a fast NestJS API backed by a Turso libSQL database.

Favourites & History

Mark games as favourites and keep a play history that syncs via JWT-secured REST endpoints. Your library is yours — stored locally and in the cloud.

Emulator Launch

One tap to hand off any title to your preferred GBA emulator via Android intent dispatch. Works with RetroArch, My OldBoy, and any emulator that accepts file intents.

Secure JWT Auth

Phone-number registration and login with bcrypt-hashed passwords. Tokens are stored in Android Keystore via flutter_secure_storage — no plain text, no Firebase dependency.

Smart Recommendations

Genre-aware suggestions based on your play history. The server analyses your favourites and surfaces titles in the same genre family you play most.

CI / CD Pipeline

GitHub Actions workflows run Flutter analyze & test, NestJS unit tests, and produce a signed APK artifact on every push to main — ship confidently.

Under the hood

From ROM list to game launch in four steps

01

Register & authenticate

Create an account with your phone number and a password. The NestJS server hashes credentials with bcrypt and issues a signed JWT stored securely in Android Keystore.

02

Browse the GBA catalogue

The Flutter app fetches paginated ROM metadata from the REST API. Titles are seeded from GBA_Roms_Names.json and GBA_Roms_Genres.json on first server startup into a Turso (libSQL) database for sub-millisecond queries.

03

Curate your collection

Favourite games are POSTed to /api/user/favourites. Your play history is tracked automatically and feeds the recommendation engine, which surfaces genre-matched suggestions in real time.

04

Launch in your emulator

Tap Play and Tsukiyomi fires an Android intent with the ROM file path. Your installed emulator (RetroArch, My OldBoy, and others) picks up the intent and launches the title immediately.
