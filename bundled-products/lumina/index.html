How it works

Four steps from recording to green tests

Lumina chains together audio transcription, frame analysis, AI reasoning, and real browser execution — all without leaving your local machine.

  1. Step 01

    Extract Audio & Frames

    FFmpeg strips audio from your Loom-style MP4. Key video frames are sampled for visual context alongside the narration track.

  2. Step 02

    Transcribe with Whisper

    Local Whisper converts spoken narration into timestamped text — no cloud API, no data leaving your machine, GPU-optional.

  3. Step 03

    Generate Test Cases via AI

    Ollama + LangChain fuse transcript and frame data to produce structured Playwright test steps — each with assertions and selectors.

  4. Step 04

    Run in Playwright

    Tests execute in a real Chromium browser — headless or headed, parallel or sequential — with screenshots captured on every failure.

Features

Everything you need to test from video

Built for developers who want CI-ready coverage without the manual scripting overhead.

  • Fully Self-hosted

    No cloud dependencies. Whisper, Ollama, and Playwright all run locally. Your recordings and tests stay on-premise.

  • Parallel Execution

    Test steps run concurrently across browser workers. Dramatically reduces wall-clock time for large test suites.

  • Screenshot on Failure

    Every failing step captures a full-page screenshot automatically, so you always know exactly what broke and where.

  • JSON Report Output

    Structured JSON reports with pass/fail status, timings, and screenshot paths — ready to integrate with CI pipelines or dashboards.

  • Desktop & Headless Modes

    Watch tests run in a visible browser window during development, then flip to headless mode for unattended CI execution.

  • Lampshade Integration

    Designed as the spotlight testing module for Lampshade — slot directly into your existing project review pipeline.
