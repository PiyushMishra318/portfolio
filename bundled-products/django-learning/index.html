What You Learn

Built to teach, built to run

Each app isolates a distinct Django skill set so you can learn one concept at a time without noise or boilerplate overhead.

JWT Authentication

Secure token-based auth via SimpleJWT. Learn access + refresh token flows, endpoint protection, and ownership-based permissions in DRF.

Syntax Highlighting API

Store code snippets with Pygments-powered highlighting. Supports 20+ languages, returned directly from the REST endpoint as styled HTML.

Session-Based Blog

Session cookies for login, user profiles with avatars, threaded comments, and profile search — classic full-stack Django without a JS framework.

SQLite — Zero Config

Both apps use SQLite out of the box. No database server to install. Run migrations and start learning in under five minutes on any machine.

Django Signals

Automatic avatar profile creation via post_save signals — a real pattern used in production Django codebases.

CI on Python 3.11–3.13

GitHub Actions runs pytest and manage.py test for both projects across three Python versions on every push.

Get Started

Running in 4 steps

No Docker, no database server, no magic. Just Python and a terminal. Each app runs independently — pick the one you want to explore first.

1

Clone the repo

Clone the monorepo from GitHub and enter your chosen project subdirectory.

git clone …
2

Create a venv

Create a virtual environment inside the project directory and activate it.

python -m venv .venv
3

Install & migrate

Install dependencies from requirements-dev.txt and run database migrations.

python manage.py migrate
4

Start the server

Launch the development server and open the browsable API or blog in your browser.

python manage.py runserver
