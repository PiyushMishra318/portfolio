Features

Everything you need to diff code

A minimal, precise, and fast comparison engine — with both a command-line interface and a web API.

  • Line-by-line comparison

    Reads both files sequentially and reports equality or difference per line — exact and unambiguous.

  • Sub-millisecond timing

    Prints elapsed time in milliseconds using std::chrono — see exactly how fast comparison runs.

  • CLI-first interface

    Pass two file paths as arguments. Exit code 0 for equal, 1 for different — fits any CI/CD pipeline.

  • HTTP REST API

    POST two snippets as JSON to /api/compare. Get back a binary result and timing — no auth needed.

  • Live browser demo

    Paste two snippets into the web UI, load equal or diff examples, and hit Compare — instant result.

  • Cross-platform build

    CMake 3.16+ with MSVC, GCC, and Clang support. Generate a Visual Studio solution or build from terminal.

How it works

Simple. Fast. Scriptable.

Three ways to use CodeDiff — pick the one that fits your workflow.

01

Build with CMake

Clone the repo and compile with CMake. The C++17 engine produces a single binary — no runtime dependencies.

cmake -B build -S .
02

Run the CLI

Pass two files as arguments. The tool prints elapsed milliseconds then 1 (equal) or 0 (different).

./build/CodeDiff a.txt b.txt
03

Call the API

POST JSON with left and right strings to the Vercel endpoint. No install, no key — just HTTP.

POST /api/compare

HTTP API

Integrate in seconds
